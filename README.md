# scraping_website
Scraping Filehippo for all of its Softwares Names and Details
-----------

" software_details_scrap.py " is supposed to bring you all the software names and links of softwares which are present on filehippo.

Right After getting those links in json file, you will send it to " software_details_scrap.py " which is responsible for grabbing all the details related to those softwares which include  Software name, software link, Software version, Software Developer, Developer Website link if there is any. 

Note that, if it does not find any information from that specific link it is given it, Python will simply write "missing" there.
